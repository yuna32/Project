## 실험 데이터 처리 과정 


 ### **1. 데이터 수집 (연구자)**
* 3~5명의 참가자(발화자)를 모집. (이들이 `Label 1`, `2`, `3`...이 됨)
* 다양한 문장(예: "안녕하세요", "불 켜줘", "오늘 날씨 어때")을 정한다.
* iOS 앱으로 각 참가자가 **모든 문장**을 여러 번 반복하여 말하도록 녹음합니다.
* 녹음 시 `.wav` (오디오)와 `.csv` (IMU) 파일이 **한 쌍**으로 저장되어야 하며 `.csv`의 `label` 컬럼에는 **'발화자 ID'** 가 정확히 기록되어야 한다.



### csv 필수 컬럼

```
1.  timestamp
- 녹음 시작부터의 시간 (초).
- 숫자 (예: `0.0`, `0.01`, `0.02`...)
- VAD(음성 감지)가 찾아낸 '시작 시간'과 '종료 시간'으로 데이터를 잘라내는 기준

2.  ax (가속도 X)
3.  ay (가속도 Y)
4.  az (가속도 Z)
5.  gx (자이로 X)
6.  gy (자이로 Y)
7.  gz (자이로 Z)
8.  label
- 발화자 ID
- 정수 (1, 2, 3)
- 이 파일이 '몇 번 참가자'의 데이터인지 알려주는 정답 값. 한 파일 안의 모든 행은 동일한 `label` 값을 가져야 한다.
```

----------------------

### **2. 데이터 준비 (연구자)**
* 모든 `.wav`와 `.csv` 파일들을 `AirSpeech/data/` 폴더(예시)에 전부 넣기
* `AirSpeech` 폴더에 저장
