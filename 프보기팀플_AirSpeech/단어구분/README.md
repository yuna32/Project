# 코드 동작

### 1. 데이터 로딩 & 신호 정제

- **입력:** `data` 폴더에 있는 모든 `.csv` 파일을 직접 읽기
- **리샘플링:** `interp1d`을 사용해 정확히 0.01초(100Hz) 간격으로 데이터를 다시 맞춘다.
- **노이즈 제거:** `Low-pass filter`를 거쳐 손떨림이나 센서의 불필요한 고주파 노이즈를 걸러낸다.

### 2. 학습용 데이터 준비

- **데이터 분리:** 전체 데이터를 80%의 훈련용(Train)과 20%의 시험용(Test)으로 나눈다. 10개 단어의 비율이 골고루 섞이도록(`stratify`) 한다.
- **표준화 :** 데이터의 값 범위를 통일한다.

### 3. 피처 추출

- 6개 축(`ax`~`gz`) 각각에 대해 통계적 요약값(평균, 표준편차, 최소, 최대, 중간값) 5개를 뽑아낸다.
- **결과:** 하나의 CSV 파일(시계열)이 30개의 숫자(6축 × 5개 통계)로 요약된다.

### 4. 모델 학습 (Random Forest)

- **입력:** 요약된 30개의 숫자들
- **학습:** 패턴-단어의 일치 규칙을 200개의 결정 트리가 학습한다.

### 5. 결과 확인

- **텍스트:** 테스트 데이터(20%)를 얼마나 잘 맞혔는지 정확도를 %로 보여준다.
- **그래프:** Confusion Matrix을 띄운다.
